# House-Rent-Prediction
Proyecto final - Modelos II

# Descripción
Este Repositorio corresponde al Proyecto final del curso de Modelos 2. Dentro de este se encuentran 2 archivos principales los cuales son: El Informe en formato Pdf en el cual se encuentra documentado el proyecto realizado y el otro archivo es el Notebook realizado para el proyecto. Este Notebook comprende todas las fases: la carga de datos, preprocesamiento, entrenamiento, evaluación y reducción de dimensión.
Es posible acceder al notebook desde Google Colab también a través de este link: https://colab.research.google.com/drive/1e93NIbvWtqQQWV-QyIA4tNHo54uROLQl?usp=sharing

# Uso del Notebook
Para usar el notebook es necesario descargar un Token desde Kaggle en Perfil -> Configuración -> Generar Token.
Luego, hay que subir dicho token al notebook bajo el nombre de kaggle.json. Esto es necesario para que sea posible leer y descargar el dataset y ejecutar el resto de celdas.
Una vez hecho eso se pueden ejecutar las celdas sin problema
