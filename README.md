# Descripción
Este Repositorio corresponde al Proyecto final del curso de Modelos 2. # Uso del Notebook
Para usar el notebook es necesario descargar un Token desde Kaggle en Perfil -> Configuración -> Generar Token.
Luego, hay que subir dicho token al notebook bajo el nombre de kaggle.json. Esto es necesario para que sea posible leer y descargar el dataset y ejecutar el resto de celdas.
Una vez hecho eso se pueden ejecutar las celdas sin problema
